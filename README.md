This is the simulation code for the paper:
"The Limits of External Manipulation in Opinion Dynamics
Through Algorithmic Targeting and the Involvement of Bots"
Alejandro Gallo Phillips, Gimnasio Campestre, Bogotá, Colombia (2026)

Requirements
pip install numpy networkx matplotlib scipy

Usage
Run `figures_all.py` in Google Colab or locally.
Generates all 4 figures from the paper.

Models
- Model A: Bounded confidence/self-assurance in opinion with stubborn agents
- Model B: Integrated SIBO + Bayesian confidence + algorithmic targeting
